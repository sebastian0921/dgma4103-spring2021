Testing Paragraphs:

This is a paragraph containing only one sentence.

This is also a paragraph. Notice the spacing between these two lines of text? That's because the paragraph tag is known as a "Block-level" tag.

Block-level tags take up ALL space available to them on the web browser/ screen the web page is being viewed on. They also usually have some "padding" added around them.

In HTML, "padding" refers to empty space surrounding the element and/or tags being used.


Testing in-line Tags/ Elements:

There are also "in-line" tags and elements. These take up only the space absolutely necessarry to display the content.

Ususally, These are links, images, and styling/ formatting tags.

Two of the most common in-line tags are em and strong.
